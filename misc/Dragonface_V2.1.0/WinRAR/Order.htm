怎样购买 WinRAR 和 RAR 许可.

如果你要在 40 天的评估过期后使用 WinRAR 和 RAR，你需要通过 中国代理 购买它的许可。你可以在 www.winrar.com.cn 查看最新的价格并在线定购.

提交了你的注册费以后，你会收到一个包含注册对应你选择的用户名字符串的 key 文件的 Email。注册时请提供一个可用的 Email 地址，它用来发送给你注册文件。

注册 email 也会包含必要的指导，只要参照执行。下面我们提供了简要的典型注册手续，但是在 Email中的指导比这个新，应该有更高的参考价值。

如果你使用 WinRAR 或 RAR/DOS, 你应该把 key 文件(rarreg.key)复制到 WinRAR 文件夹中。默认的文件夹是“C:\Program Files\WinRAR”,但是它能在安装 WinRAR 时被用户改变。

如果这个 key 是 RAR 压缩文件，为了注册你的 WinRAR,请从压缩文件中解压 key 。

使用 RAR/Unix 和 RAR for OS X，你应该把 rarreg.key 复制到你的 home 目录或下列目录之一: /etc, /usr/lib, /usr/local/lib, /usr/local/etc。你可以把它改名为 .rarreg.key 或 .rarregkey。如果你愿意，rarreg.key 的名字也是可以的。

现在， WinRAR， RAR for Unix, DOS 和 OS X 使用同样的注册格式，所以你可以使用当前 WinRAR 和 RAR 版本的同一个注册码用于多个平台。
